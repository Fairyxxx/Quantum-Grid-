Quantum Grid

v1.0.0 MIT License 纯前端 零依赖构建

高性能浏览器端二维码生成器。基于 Canvas API 实时渲染，支持多级纠错、PNG 下载与剪贴板复制。粒子背景动画，终端美学设计。

功能特性

  • 实时生成，输入即所得
  • 四级纠错可选（L / M / Q / H）
  • 一键下载 PNG（240×240）
  • 一键复制到剪贴板
  • 暗色终端工业美学
  • 粒子动态背景
  • 响应式布局，移动端适配

快速开始

直接在浏览器中打开 index.html：

https://github.com/Fairyxxx/Quantum-Grid-.git
cd quantum-grid
open index.html

或在输入框中键入文本 / URL，二维码即时生成。

技术栈

  • qrcode — 轻量 QR 生成库
  • Canvas API — 高速位图渲染
  • 纯 CSS 动画与粒子系统
  • Clipboard API — 剪贴板写入

参数说明

纠错级别：L ≈ 7% / M ≈ 15% / Q ≈ 25% / H ≈ 30% 容损率。级别越高，二维码越密集，但抗损坏能力越强。默认推荐 H。
